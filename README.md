## üöÄ Web Hosting & Domain Explanation

As part of the assignment requirements, here is the breakdown of the hosting and domain strategy used for this project:

### 1. Web Hosting: Vercel
I chose **Vercel** to host this website.
* **Reasoning:** Vercel offers a seamless integration with GitHub. It automatically deploys the website whenever I push changes to the code repository (Continuous Deployment).
* **Reliability:** It utilizes a global Content Delivery Network (CDN), ensuring the site loads quickly for visitors regardless of their location.

### 2. Domain Name: `keiths-portfolio.vercel.app`
* **Identity:** For this project, I secured the unique subdomain `keiths-portfolio` under the Vercel domain. This clearly represents my personal brand ("Keith") and the purpose of the site ("Portfolio").
* **Future Expansion:** In a professional production environment, I would register a custom Top-Level Domain (TLD) such as `emmanuelkiio.tech` or `stargenius.dev` through a registrar like Namecheap or GoDaddy to further solidify the brand identity.
